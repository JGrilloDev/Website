Global Game Jam 2020

The theme of this years Global Game Jam was "Repair". I was working in a team of just myself and another programmer, Paul. The majority of programming we did on this project was as a pair, as we have different pockets of knowledge. When the game had been built up enough that we were happily able to work on different areas, my focus mainly on building what we needed for UI, such as buttons and the functionality to move between scenes.
